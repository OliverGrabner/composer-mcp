# MCP Directory Submissions

Track submissions to MCP server directories for discoverability.

## Tier 1 — Highest Impact

- [ ] **awesome-mcp-servers** — PR to [punkpeye/awesome-mcp-servers](https://github.com/punkpeye/awesome-mcp-servers)
  - Category: Developer Tools (alphabetical order)
  - Format: `- [Composer](https://github.com/olivergrabner/composer-mcp) — Design and visualize software architecture diagrams with AI. Give your coding agent an interactive canvas for system design.`
  - Read [CONTRIBUTING.md](https://github.com/punkpeye/awesome-mcp-servers/blob/main/CONTRIBUTING.md) before submitting
  - Include emoji tags per their legend

- [ ] **Official MCP Registry** — [registry.modelcontextprotocol.io](https://registry.modelcontextprotocol.io/)
  - Namespace: `io.github.olivergrabner/composer-mcp`
  - Requires GitHub ownership verification

## Tier 2 — Good Visibility

- [ ] **Smithery.ai** — [smithery.ai](https://smithery.ai/)
  - CLI: `smithery mcp publish "https://github.com/olivergrabner/composer-mcp"`
  - Or submit via web interface

- [ ] **mcpservers.org** — [mcpservers.org/submit](https://mcpservers.org/submit)
  - Web form submission (no PRs accepted)

- [ ] **mcp.so** — [mcp.so](https://mcp.so)
  - Community submission

## Tier 3 — Auto-indexed (no action needed)
